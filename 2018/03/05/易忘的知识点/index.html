HTML、CSS中的易忘知识点

HTML部分

  1. a标签的target属性：用于指定链接打开的方式。_self为默认值，_blank为在新窗口中打开方式。

  2. 锚点定位：①. 同页面定位

    ​ ②. 不同页面定位(前面加上路径即可)

  3. base 可以设置整体链接的打开状态

    nofollow设置不抓取链接

  4. 表格属性table

    ​ cellspacing：设置单元格与单元格的边框之间的空白间距

    ​ cellpadding：设置单元格内容与单元格边框之间的空白间距

    ​ caption：表格标题(标签必须紧随table标签之后)

    ​ rowspan：跨行合并

    ​ colspan ：跨列合并

CSS部分

  1. background：color url(不推荐加引号) repeat position attachment(scroll | fixed)

  2. 清除浮动方法：

    ①额外标签法：在浮动元素末尾加一个空的标签

    ②给父级添加overflow属性方法：通过触发BFC，实现清除浮动

    ③使用after伪元素清除浮动

    ④after伪元素空余字符法：content:”\200B”;这个参数，Unicode字符里有一个“零宽度空格”，即 U+200B，代替原来的“.”，可以缩减代码量。而且不再使用visibility:hidden。

    ⑤使用before和after双伪元素清除

  3. 元素的显示与隐藏

    ​ ①display：none | block

    ​ 特点：隐藏之后，不在保留位置

    ​ ②visibility：visible | hidden

    ​ 特点：隐藏之后继续保留位置

    ​ ③overflow： visible : 　不剪切内容也不添加滚动条。

    ​ auto : 　 超出自动显示滚动条，不超出不显示滚动条

    ​ hidden : 　不显示超过对象尺寸的内容，超出的部分隐藏掉

    ​ scroll : 　不管超出内容否，总是显示滚动条

  4. 防止拖拽文本域resize

    resize：none

  5. vertical-align垂直对齐

    ​ vertical-align : baseline |top |middle |bottom

    ​ vertical-align 不影响块级元素中的内容对齐，它只针对于 行内元素或者行内块元素，特别是行内块元素， 通常用来控制图片和表单等。

  6. CSS文本属性

    ​ 字间距：letter-spacing

    ​ 单词间距（英文单词）：word-spacing

    ​ 自动换行：word-break：normal | break-all（允许在单词内换行）| keep-all（只能在半角空格或连字符处换行）

    ​ 空白符处理：white-space：normal | pre(预格式化) | nowrap(空格空行无效，强制文本不能换行，除非遇到换行标记
    。内容超出元素的边界也不换行，若超出浏览器页面则会自动增加滚动条)

  7. 文字溢出：text-overflow ： clip(不显示省略的标记，裁切) | ellipsis(显示省略的标记…)

  8. 兼容IE
