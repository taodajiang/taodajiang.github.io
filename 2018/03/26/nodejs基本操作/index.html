Node.js

  • Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。
  • Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。
  • Node.js 的包管理器 npm，是全球最大的开源库生态系统。

常用模块API

Buffer

本质上是字节数组，Node处理二进制数据的一个API。Node原生提供的全局对象，可以直接使用，不需要require('buffer')。

  1. Buffer实例化
    1. Buffer.alloc(size)
    2. Buffer.alloc(string)
    3. Buffer.from(array)
    4. Buffer.from(string)
  2. Buffer静态方法
    1. Buffer.isEncoding() 判断是否支持该编码
    2. Buffer.isBuffer()判断是否为Buffer
    3. Buffer.byteLength()返回指定编码的字节长度，默认为utf-8
    4. Buffer.concat()将多个Buffer对象合并为一个Buffer
  3. Buffer 实例方法
    1. write() 向Buffer中写入内容
    2. slice()截取，两个参数，与js相同，包头不包尾
    3. toString()把Buffer转换成字符串

路径操作 path

windows是右斜杠，Liux是左斜杠

path模块，需要引入const path = require('path');

  1. path.basename( ) 返回路径中的最后一部分，即文件名称
  2. path.dirname( ) 获取路径
  3. path.extname( ) 获取扩展名称
  4. 路径的格式化处理
    1. path.format( ) obj ->string
    2. path.parse( ) string -> obj
  5. path.isAbsolute( ) 判断是否为绝对路径
  6. path.join( pathstring1,pathstring2… ) 拼接路径，在连接路径的时候会格式化路径，进行解析
  7. path.normalize( pathstring ) 规范化路径
  8. path.relative( pathstring1,pathstring2 ) 计算相对路径
  9. path.resolve( ) 用来解析路径
  10. 两个特殊属性
    1. path.delimiter 路径分隔符（windows是\,Linux是/）
    2. path.sep 环境变量分隔符（windows是; Linux是:）
